    1.浏览器

    之前用的是Chrome，现在改用Edge，

    Microsoft Edge

    基于chrome内核，在windows操作系统内优化不错，相比于chrome内存占用低，界面美观，且可以直接使用chrome应用商店下载并兼容所有插件

    • 在用插件
      • Chrome商店：
        • Google 翻译
        • Infinity 新标签页
        • IDM下载器：IDM Integration Module
        • 免费的广告拦截器：Adblock Plus
        • Steam市场、库存插件：Steam Inventory Helper
        • CS:GO市场磨损、印花查看插件：CSGOFloat Market Checker
      • 油猴Tampermonkey：
        • 中国大学慕课mooc答题/自动播放脚本(domooc)
        • 百度文库（wenku）在线下载PDF格式文件
        • 爬取页面视频、音乐：音视频通用下载器
        • 网易BUFF市场插件---CSGO版
        • 计算国内交易平台与Steam市场价格比例：饰品比例计算脚本
        • 多平台价格对比：CSGO饰品价格对比脚本

    2.下载器

    迅雷

    只负责磁力链接和BT下载。迅雷虽然口碑很差，但不管怎么说，在国内这两种下载还是用它最快(敏感资源另说)

    Internet Download Manager

    目前个人最常用的下载器，负责磁力和BT外所有下载任务，界面相当朴实无华（简陋）

    图床

    picsum(可以美化图片的随机图床)

    • 官方网站

    PicGo-图床工具

    挺火的一个图床管理工具，结合github+jsdelivr可以白嫖cdn加速的高速免费图床，就是实际使用中经常抽风导致图片上传失败，那时候就得自己直接去github上传，然后图片url加上jsdelir的加速域名

    阿里云-对象存储OSS

    新号一年10块钱40g容量，用于存储图片还可以，但如果用于搭建国内无法备案的网站（听说可以买别人的备案服务码申请备案，但是很容易被查出来然后域名被封…有点想直接买个国内的云服务器了，最近几个大厂都有活动100块钱就能买一年），因为无法享受CDN服务，流量费用（0.5元/G）太贵了！
