软件、网站使用笔记
发表于 2020-04-18| 更新于 2020-04-18
阅读量:|评论数:

软件

1.浏览器

之前用的是Chrome，现在改用Edge，

Microsoft Edge

基于chrome内核，在windows操作系统内优化不错，相比于chrome内存占用低，界面美观，且可以直接使用chrome应用商店下载并兼容所有插件

在用插件

  • Chrome商店：

    • Google 翻译
    • Infinity 新标签页
    • IDM下载器：IDM Integration Module
    • 免费的广告拦截器：Adblock Plus
    • Steam市场、库存插件：Steam Inventory Helper
    • CS:GO市场磨损、印花查看插件：CSGOFloat Market Checker

  • 油猴Tampermonkey：

    • 中国大学慕课mooc答题/自动播放脚本(domooc)
    • 百度文库（wenku）在线下载PDF格式文件
    • 爬取页面视频、音乐：音视频通用下载器
    • 网易BUFF市场插件---CSGO版
    • 计算国内交易平台与Steam市场价格比例：饰品比例计算脚本
    • 多平台价格对比：CSGO饰品价格对比脚本

2.下载器

迅雷

只负责磁力链接和BT下载。迅雷虽然口碑很差，但不管怎么说，在国内这两种下载还是用它最快(敏感资源另说)

Internet Download Manager

目前个人最常用的下载器，负责磁力和BT外所有下载任务

3.开发工具

VSCode

在用插件

  • 全局：

    • VSCode汉化：Chinese (Simplified) Language Pack for Visual Studio Code
    • 智能填充：Kite Python Autocomplete
    • 状态栏显示文件大小：filesize

  • Python:

    • python

  • Markdown:

    • Markdown All in One(很多人推荐，但我还没研究透怎么用)
    • 强化预览功能，并可用来插入图片：Markdown Preview Enhanced
